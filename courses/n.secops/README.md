## Curso: n.secops

Este diretório serve como **scaffold versionado** para organizar a criação do curso com slug **`n.secops`**.

### Arquivos principais
- `course.md`: visão geral (objetivos, público, pré‑requisitos, carga)
- `syllabus.md`: ementa (módulos, tópicos e trilha sugerida)
- `programa.md`: baseline do programa (estrutura macro)

### O que entra aqui
- **Roteiro/ementa** do curso (módulos, aulas, objetivos, requisitos).
- **Referências** (links, normas, materiais base).
- **Notas de produção** (ex.: o que falta gravar, revisar, validar).

### O que NÃO muda no app
Este diretório **não é lido pelo Next.js** e **não afeta runtime**. É apenas organização/documentação no repositório.

### Como “criar” o curso no sistema (resumo)
1. No painel admin: `Admin > Courses > New`
2. Preencher o **Slug** exatamente como: `n.secops`
3. Criar **módulos** e **aulas** (vídeo/texto/pdf/quiz) via UI existente
