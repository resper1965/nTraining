## Módulo 07 — Incident Response (IR)

**Duração estimada:** 2h  
**Objetivo:** conhecer o processo estruturado de resposta a incidentes, playbooks e governança de comunicação.

### Tópicos
- Frameworks (NIST, SANS)
- Fases: Preparação → Detecção → Contenção → Erradicação → Recuperação → Lições Aprendidas
- Playbooks e runbooks
- Escalação e comunicação (técnico/gestão)
- Preservação de evidências
- War room e gestão de crise
- Integração com n.cirt

### Exercício (sugestão)
Executar um “tabletop” de incidente (phishing com credenciais) usando:
- `templates/incidente.md`
- critérios de escalonamento e comunicação
