## Quiz — Módulo 03 (SIEM: Correlação de Eventos)

- A) Aumentar o tamanho do log para auditoria
- B) Padronizar campos/formatos para permitir correlação e triagem consistente
- C) Remover timestamps para evitar inconsistências
- D) Substituir o SIEM por dashboards manuais

- B) Definir limites (quantidade/janela/condição) para disparar alerta com menos ruído
- C) Evitar necessidade de fontes múltiplas
- D) Eliminar falsos positivos automaticamente

3) O que mais melhora a “acionabilidade” de um alerta?
- A) Uma descrição genérica e curta
- B) Evidências mínimas no payload (user, host, IP, timestamps, ação)
- C) Um gráfico bonito no dashboard
- D) Aumentar severidade de tudo para “alto”

4) Qual item é típico de um dashboard operacional (turno)?
- A) Tendência anual de orçamento
- B) Fila por severidade e idade (aging) + SLA
- C) Roadmap de produto
- D) Lista de normativas ISO completas

5) Qual é uma prática recomendada ao especificar regra “password spraying”?
- A) Não excluir nada para não “perder” ataques
- B) Definir janela/contagem e considerar contexto (conta privilegiada, IP novo) + exclusões conhecidas
- C) Disparar alerta em qualquer falha isolada

3) B
